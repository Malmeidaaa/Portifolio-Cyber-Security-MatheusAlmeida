CompTIA Security+ Laboratories

The CompTIA Security+ course on the Cengage platform provided a very rich practical experience through virtual laboratories integrated into MindTap, using virtual machines in a secure "sandbox" environment. In each lab, it was possible to apply essential security concepts practically—such as network protection, vulnerability analysis, and incident response—strengthening both theoretical knowledge and the technical skills required by the Security+ certification.

Throughout the labs, I worked with virtualized Windows and Linux operating systems, executing tasks such as firewall configuration, network traffic analysis, and the use of scanning and threat detection tools, always on pre-configured servers simulating real IT scenarios. This format allowed exploring different solutions, making mistakes, and trying again without risk to production environments, which is fundamental for developing confidence in security decision-making.

The laboratories also addressed topics such as cryptography, identity management, and access control, aligned with the official Security+ exam domains. This helped consolidate content such as CIA (confidentiality, integrity, and availability), AAA, and security controls. Overall, these practical exercises were decisive in connecting theory and practice, better preparing for the exam and for real challenges in the cybersecurity field.
